### 1.7
- word : 空白で区切られた、文字・数字・アンダースコアからなる文字列
- WORD : 空白で区切られた、空白以外の全ての文字からなる文字列

| キー | 動作 |
| w | word単位で前に進む |
| e | wordの末尾へ前に進む |
| W | WORD単位で前に進む |
| E | WORDの末尾へ前に進む |
| b | wordの先頭へ後ろに進む |
| B | WORDの先頭へ後ろに進む |
| { | 段落単位で後ろに戻る |
| } | 段落単位で前に進む |
| cc | 行を丸ごと削除してインサートモード |
| dd | 行を丸ごと削除 |
