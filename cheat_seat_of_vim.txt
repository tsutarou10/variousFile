w -> 次の単語の先頭
e -> 今の単語の末尾
b -> 前の単語の先頭
% -> 対応する括弧に
gg -> ファイルの先頭へ
G -> ファイルの最後へ
:数字 -> 数字で指定された行へ
C-r -> uで取り消したものを戻す
dd -> カーソル上の行全体をカット
D -> カーソル位置から行末までカット
daw -> カーソル上の単語全てをカット
yy -> カーソル上の行全体をヤンク
p -> ヤンクした文字を貼り付け

======
(挿入モード)
I -> カーソル行の先頭から
a -> カーソルの後ろの文字から
A -> カーソル行の末尾から
s -> カーソル上の文字を削除してその場所から
S -> カーソル上の行全体を削除してその場所から
o -> 次の行を追加してその行から
O -> 前の行を追加してその行から
R -> 上書きモードへ

=======
(visualモード時)
x -> 削除
d -> カット
y -> ヤンク
gu -> 小文字化
gU -> 大文字化
> インデントを深くする
< インデントを浅くする
= オートインデント

その他
:set number -> 行数
:set hlsearch -> 検索後ハイライト表示
:set tabstop=数字 -> タブ幅の指定
n -> 検索モードで次の文字に移動
N -> 検索モードで逆方向に移動
:%s/"置換前"/"置換後" -> 各行の一番初めの文字列を置換
:%s/"置換前"/"置換後"/g -> 一行中に複数マッチしても置換
zo, zO -> oで1段階, Oで全て開く
zc, zC -> cで1段階, Cで全て閉じる 
ヴィジュアルモードでzf -> マーク埋め込み
zd -> マーカー削除
